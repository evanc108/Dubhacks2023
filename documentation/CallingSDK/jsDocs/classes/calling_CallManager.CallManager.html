getCallById

  • getCallById(callId: string): Call

  • Get a specific active call by Id value. First looks for a matching networkId value, and then if not found looks for a matching localId value. The localId is generated when the call is first created locally. When the call is started, it will be assigned a network id value. Some code might still have the localId value, so both values are kept so all references can be match. In-coming calls will always have localId and networkId be the same value. Only out-going calls with have separate values for localId and networkId.

    Parameters

    • callId: string

    Returns Call
