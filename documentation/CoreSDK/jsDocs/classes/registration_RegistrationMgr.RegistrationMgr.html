appStart

  • appStart(): Promise<ConnectionState>

  • appStart Determines if this is a cold start, or a redirect back from sign-in, or a warm restart. Then performs necessary steps to get app correctly initialized. For Cold Start: There is nothing to do other than set the current app state to 'new'. For Return from Redirect: extracts the authCode value from query string parameters, and exchanges it for a CPaaS Auth Token, then initializes the system. For Warm Restart: Recovers the saved CPaaS Auth Token, then initializes the system. Must be connected to the Internet to succeed.

    Returns

    the current app connection state.
